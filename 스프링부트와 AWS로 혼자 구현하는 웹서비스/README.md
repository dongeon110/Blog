# 스프링부트와 AWS로 혼자 구현하는 웹서비스  
(by 이동욱, publishing by 프리렉)

### 요구사항 분석  
3장에서 6장까지 하나의 게시판(웹 애플리케이션)을 만들어 보고 7장부터 10장까지는 이 서비스를 AWS에 무중단 배포 하는것까지 진행합니다.  

이 게시판의 요구사항은 다음과 같습니다.  
```
- 게시판 기능
1. 게시글 조회
2. 게시글 등록
3. 게시글 수정
4. 게시글 삭제
```
```
- 회원 기능
1. 구글/네이버 로그인
2. 로그인한 사용자 글 작성 권한
3. 본인 작성 글에 대한 권한 관리
```

위 기능들을 앞으로 하나씩 만들어 가면서 제가 생각하는 필요한 기능,  
예전에 [Servlet으로 만든 게시판](https://github.com/dongeon110/Making_Web_Board)에서 개선할 점 들을 적용시켜가면서 만들어 보겠습니다.
