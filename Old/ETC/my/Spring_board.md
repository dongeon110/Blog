## page 기능
```
　┬ /board/
　│　├ list.do   (게시판) √
　│　├ view.do   (게시물 보기)
　│　├ update.do   (게시물 수정)
　│　├ delete.do   (게시물 삭제)
　│　├ repost.do   (답글 달기)
　│　└ add.do   (게시물 추가)
　│
　├ /auth/
　│　├ login.do   (로그인)
　│　└ logout.do   (로그아웃)
　│
　└ /user/
 　　├ list.do   (사용자 관리)
 　　├ add.do   (회원가입)
 　　├ delete.do    (사용자 삭제)
 　　└ update.do    (사용자 수정)
```
