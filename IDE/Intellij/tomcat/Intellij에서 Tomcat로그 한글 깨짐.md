# 인텔리제이에서 톰캣 로그 한글 깨짐 현상 수정  

![깨진상태](./images/0.%ED%95%9C%EA%B8%80%EA%B9%A8%EC%A7%90_%EA%B9%A8%EC%A7%84%EC%83%81%ED%83%9C.PNG)  

인텔리제이에서 톰캣으로 서버 실행시 로그가 한글이 깨져서 출력 되었습니다.  

톰캣 설정을 변경해도 계속 한글이 깨져서 인텔리제이에서 방법을 찾게 되었습니다.  

## 방법  

1. 검색창을 켜고 "Edit Custom VM Options"을 검색하고 선택합니다.  

검색창은 Ctrl + Shift + A 키를 눌러서 열 수 있으며 "액션" 탭에서 검색합니다.  
![검색창](./images/0.%ED%95%9C%EA%B8%80%EA%B9%A8%EC%A7%90_%EA%B2%80%EC%83%89%EC%B0%BD.PNG)  

한글 설정이 되어 있으면 "사용자 지정 VM 옵션 편집" 을 검색해도 되고, 영어로 검색하더라도 나옵니다.  

2. 가장 밑에 아래 설정을 하나 추가합니다.  
```-Dfile.encoding=UTF-8```  
![인코딩추가](./images/0.%ED%95%9C%EA%B8%80%EA%B9%A8%EC%A7%90_%EC%9D%B8%EC%BD%94%EB%94%A9%EC%B6%94%EA%B0%80.PNG)  

3. 인텔리제이를 재실행하고 톰캣을 실행해봅니다.  

![해결](./images/0.%ED%95%9C%EA%B8%80%EA%B9%A8%EC%A7%90_%ED%95%B4%EA%B2%B0.PNG)  

한글이 정상적으로 출력되는 것을 확인했습니다.
