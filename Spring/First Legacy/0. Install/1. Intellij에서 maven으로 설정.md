# Intellij에서 maven으로 설정하기

Intellij Community 버전은 스프링 프레임워크를 지원하지않아서  
직접 설정하고 사용해야함.  
Maven + Spring + Tomcat 진행중  

## 새 프로젝트 생성  
인텔리제이 메뉴 창에서 **\[파일\] - \[새로 만들기\] - \[프로젝트\]** 를 선택해서 새 프로젝트를 생성합니다.  

![프로젝트생성](./images/0.1_%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8%EC%83%9D%EC%84%B1.PNG)  

이름은 원하는 프로젝트 이름으로 설정하였습니다.  
위치는 프로젝트가 설치될 위치이며,  
Java와 Maven을 사용해서 Spring을 적용하겠습니다.  

JDK의 경우에는 현재 최신버전인 Spring5 버전이 Java8을 표준으로 사용하고 있기 때문에 8을 사용하겠습니다.  
(1.8과 8은 같다고 봐도 무방합니다.)  

그리고 프로젝트를 생성합니다.  

![프로젝트생성완료](./images/0.1_%ED%94%84%EB%A1%9C%EC%A0%9D%ED%8A%B8%EC%83%9D%EC%84%B1%EC%99%84%EB%A3%8C.PNG)  

프로젝트가 생성되면 왼쪽에 위 사진 처럼 프로젝트가 생성됩니다.  

이제 스프링 프레임워크를 적용시켜보겠습니다.  

pom.xml을 이용하여 설정을 하도록 하겠습니다.  

초기에 프로젝트를 생성했을 때 생성된 pom.xml은 아래와 같은 형태를 가지고 있었습니다.
